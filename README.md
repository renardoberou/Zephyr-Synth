# Zephyr Synth

This build adds a usable modulation system and macro control.

## Current features

- 3 oscillators
- waveform selection per oscillator
- detune per oscillator
- oscillator mixer
- low-pass filter
- resonance
- ADSR amplitude envelope
- LFO source
- 2 macro sources
- 4-slot modulation matrix
- destinations:
  - oscillator detune
  - filter cutoff
  - amp level
  - chorus mix
  - delay mix
  - reverb mix
- chorus
- tape-style delay
- reverb
- on-screen keyboard
- AudioWorklet oscillator
