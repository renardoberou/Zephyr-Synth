# Zephyr Synth

Larger-step build: subtractive core plus modulation and FX.

## Current features

- 3 oscillators
- waveform selection per oscillator
- detune per oscillator
- oscillator mixer
- low-pass filter
- resonance
- filter envelope amount
- ADSR amplitude envelope
- global LFO with:
  - pitch modulation
  - filter modulation
  - amplitude modulation
- chorus
- tape-style delay
- reverb
- on-screen keyboard
- AudioWorklet oscillator
